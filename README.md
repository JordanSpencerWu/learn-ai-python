# Learn AI Python

A demo project to learn OPENAI in python.

## Getting Started

- `brew install ffmpeg`
- `asdf install`
- `pip install -r requirements.txt`
- `python setup.py`
- Set `OPENAI_API_KEY` in `.streamlit/secrets.toml` file
- `streamlit run Hello.py`

## Automatically create `requirements.text`

Run `pipreqs`
